Web Submission and Review Software

September 2009

Shai Halevi

This is the information page for the Web Submission and Review Software that was written by Shai Halevi from IBM Research. This is a conference management software that provides support for the entire life-cycle of the conference review process, starting with the submissions of papers and ending in the receipt of the camera-ready versions of the accepted papers (and even automatic generation of the table-of-contents and author index for the conference proceedings).

What you can find here:

  • The latest version of the software (currently 0.63).
  • User-guides for the program chair, program-committee members, and authors.
  • A demo (made enitrely of static HTML).
  • A small testing package for experimenting with the software.
This software is open source, and it is distributed under the terms of the open-source Common Public License (CPL) v1.0. See the terms in the file LICENSE.txt in this package or at http://www.opensource.org/licenses/cpl1.0.php. I now have a sourceforge project for it at http://sourceforge.net/projects/websubrev.

One of the goals of this software was to make administration as easy as possible, so you will not need to run scripts etc. In principle, to install this software you just unpack the zip file in some directory in your web tree (call it the BASE directory), point your browser at that directory and off you go. However, this simple method of installation requires the web-server to be able to write files in the web-tree, which in some cases is considered a security risk. I therefore provide also a non-web-based installation method. See more information in the program chair documentation.

I will very much appreciate any comments that you can give me regarding this software. Please contact me at shaih@alum.mit.edu. I am very grateful to the many people who gave me feedback on this software.
